# Global University Rankings Analysis

This project analyses the QS world rankings using python and pandas to explore and shortlist universities across different countries.

# Features

- Clean and process CSV data
- Filter universities by target countries
- Calculate summary statistics (mean, median, minimum, maximum)
- Identify top 10 universities by country
- Extract insights from rankings and scores

# Skills Demonstrated

- Python programming
- Data cleaning with pandas
- Exploratory data analysis
- Sorting, filtering, and grouping data
- Basic statistical analysis
